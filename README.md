# 🕹️ teamzcompany

Bienvenue sur **teamzcompany** — un portfolio pas comme les autres, où chaque page cache son lot de surprises !

## ✨ Présentation

Ce site statique (HTML, CSS, JS pur) est bien plus qu’un simple portfolio :  
c’est un terrain de jeu interactif rempli d’animations originales et de combinaisons secrètes à découvrir… si tu es assez curieux !

> **Explore, clique, tente, expérimente…**  
> Les plus observateurs et les explorateurs dans l’âme trouveront des easter eggs cachés partout !

---

## 🔥 Points forts

- **Animations stylées** : transitions fluides, effets interactifs, visuels soignés
- **Easter eggs** : sauras-tu tous les trouver ? (Certaines combinaisons de touches ou d’actions déclenchent des surprises.)
- **Code 100% natif** : pas de frameworks, juste du HTML, CSS et JS pour une expérience rapide et légère

---

## 🚀 Découvrir

Clone le repo, ouvre `index.html` et amuse-toi à tout tester.  
Chaque interaction pourrait cacher un secret…

```
git clone https://github.com/zay168/teamzcompany.git
cd teamzcompany
start index.html
```

---

## 🎯 Objectif

Montrer qu’un site statique peut être **dynamique, créatif et fun** — sans dépendance externe.

---

## 👾 Un indice ?

Rien n’est laissé au hasard… **Les combinaisons de touches, les clics et même les éléments cachés sont vos alliés.**  
Bonne chasse aux secrets !

---

*Merci de ne pas spoiler dans les issues 🙏*
